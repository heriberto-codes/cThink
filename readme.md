## What is cThink?
cThink is a multi-sensory learning tool that uses a learning assessment and the D.A.B. method ‚Äî Draw, Act, Build ‚Äî to help learners simplify technical concepts and retain them through association building.

Inspired by real classroom experiences and lessons with his young son Elijah, Software Engineer and technical instructor Heriberto Roman set out to address a common challenge: learners of all ages struggling to process and retain abstract concepts.

cThink tackles this challenge by making the learning process visual, auditory, and kinesthetic ‚Äî accelerating understanding and boosting memory retention.

## How It Works
